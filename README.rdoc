#ACD Viewer#

##Dependencies Used

* **Easy Drop Down** : <http://patrickkunka.github.io/easydropdown/>
* **Ajax Loader** : <http://preloaders.net/>
* **Vague JS** : <https://github.com/GianlucaGuarini/vague.js/>
* **Ruby** : ruby '1.9.2-p320'
* **Rails** : Rails 3.2.16

##Running the Application

1. Checkout the ACD Viewer repository from Github
2. '**cd**' to the directory
3. Run **rails s**
4. You will be able to view the application from 
```
http://localhost:3000/acd_viewer
```
